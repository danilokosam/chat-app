# Chat App

Aplicación de chat en tiempo real que permite a los usuarios comunicarse de forma instantánea.

## Descripción

Esta es una aplicación web que facilita la comunicación entre usuarios mediante mensajería instantánea en tiempo real.

## Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: (Por definir)

## Instalación

```bash
# Frontend
cd frontend
npm install

# Backend
cd backend
npm install
```

## Uso

```bash
# Ejecutar frontend
cd frontend
npm run dev

# Ejecutar backend
cd backend
npm start
```
